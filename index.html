let originalobj = {
name: 'Ronak Koladiya',
email: 'ronak@info.com',
number: '9087645673'
address:{
city: 'Surat',
state: 'Gujarat'
}
};

let copyobj = originalobj;

let spreadobj = {...originalobj};

EXPLANATION

1) In this part we have one original object set with name, email, number and address, Now we copied one object on which we have just assign a direct values of original object directly like:(copyobj = originalobj) now after entering text in first input type when user clicks on submit we just changed the value of normal copy's name but as mentioned we directly assigned a value to the copied object because of that it creates the shallow copy of original object that means if we change only copied object value it also changes the value of original object as well.

2) We created a new object and assigned the value with spread operator as show above like:(spreadobj = {...originalobj}) now on second input type we are changing the name's value of spread copy object if user clicks on submit after entering the data in second input type, In spread copy it won't change the name because its a part of main data in object its only changes spread copy's name its kinda work like deep copy but only till some stage we gonna learn next stage in next example.

3) Now we added a sub data in object with address there we have city and state, Now we gonna change the city's value on click of third input type submit, after entering value and submitting we can see that it changes all values of city because now the city is under the sub category of object the assigned value of spread object is only gonna work till main category of object after that it also creates the shallow copy of sub data.
